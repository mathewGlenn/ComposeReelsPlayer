# GitHub Pages

This folder contains the Jekyll site for Compose Reels Player.

To publish it with GitHub Pages:

1. Open the repository settings on GitHub.
2. Go to **Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Select the default branch and the `/docs` folder.
5. Save the setting.

The site is intentionally built with GitHub Pages' default Jekyll support: no custom plugins, gems, or GitHub Actions workflow are required.
